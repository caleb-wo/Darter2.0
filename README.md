# Darter2.0

## Intro

The game _will_ become a simple 2D platformer about a dart frog exploring a jungle. It's a pixel game & the frog will be able to shoot poisonous darts at enemies.

## Tools

- [Excalibur.js](https://excaliburjs.com), an open source, typescript first, HTML5 game engine
- Typescript
- Bun & Vite
- **Front-end Framework (Future):** Angular (planned for site and UI integration)

## Demo

The following was generated by Bun, but should work.

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run serve
```

This project was created using `bun init` in bun v1.3.1. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
